Installing phpLinkDropper

Follow the steps below to install your copy of phpLinkDropper

1. Create a MySQL database on your site to use with phpLinkDropper.

2. EDIT the config.php with your database information (username, password and database name).

3. Upload all the files in the phpLinkDropper distribution to your web server in the directory you wish to run it from. PLEASE MAKE SURE TO KEEP THE
DIRECTORY STRUCTURE INTACT ... OTHERWISE YOUR WEBSITE WILL NOT FUNCTION. You can run the script from any directory you wish on your web server, but the scripts directory structure must stay the same. (i.e. http://www.yourdomain.com/ or http://www.yourdomain.com/links/ or even http://www.yourdomain.com/linktown/links/)

4. Run install.php via your web browser.

5. If the install process has worked, you should be able to login to your admin area at http://www.yoursite.com/admin/index.php using the username admin and password admin.

6. Now you will need to go into the Site Configuration portion of the control panel and make sure to enter in the base url for your site. Make sure to keep
the trailing / after your domain name (i.e. http://www.yourdomain.com/)

7. If you plan on turning SE Friendly URLs ON (default is set to ON), make sure to rename the file htaccessfile to .htaccess

8. IF YOU PLAN ON USING the prune option to delete content that is older than a certain amount of time, then you will need to setup a cronjob to run the file /admin/nightly.php. This file will check the database for any items older than your prune date and remove them. This file also is useful because it will do nightly maintenance on your MySQL database to make sure it is running fine.

Congratulations! Your new phpLinkDropper site is setup and ready to go!
